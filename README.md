Overview
This project is a smart contract that allows users to stake Ethereum (ETH) and receive Trump Coin as a reward. The contract ensures secure staking with protection against reentrancy attacks and can be paused if needed.

Features
✅ Stake ETH – Users can deposit ETH into the contract.
✅ Earn Trump Coin – Stakers receive Trump Coin as a reward for staking.
✅ Secure & Optimized – Uses reentrancy protection and follows best Solidity practices.


How It Works
Users call the stake function and send ETH.
The contract records the staked amount and updates the user’s balance.
Over time, users accumulate Trump Coin based on the staking rewards logic.
Users can later withdraw rewards or their original stake when the conditions allow.
Technologies Used
Solidity – Smart contract programming.
Remix IDE – For deployment and testing.
Ethereum Blockchain – The network for transactions.
ERC-20 (Trump Coin) – Token standard for rewards.
